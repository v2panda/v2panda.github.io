前言

这是本博客的第一篇文章，用来记录这个网站的诞生过程。

环境

Required

  • OS X
  • HomeBrew
  • Node.js
  • Git
  • Hexo
  • Github

Optional

  • 百度统计账户
  • 域名
  • DNSPod账户
